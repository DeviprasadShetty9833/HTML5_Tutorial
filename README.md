# HTML

‚Ä¢ Extension :  `.html`  OR  `.htm`

‚Ä¢ Syntax of Html : 
```html

<html>
  <head>
    <title> TITLE </title>
  </head>

  <body> Actual contents in the page
  </body>
</html>

```
---

‚û¢ Attributes of `<body>` tag :

> - Background - image file as a background behind the displayed text.
> - BGCOLOR - background color of web page.
> - Text - text color 
> - Link - color for hyperlinks.
> - Vlink- color of visited hyperlinks.

‚óè Paired Tags

For Bold : 
```html

```

For Italics : 
```html
<I> ... </I>
OR

```

For Underline : 
```html

```

For Big & Small font  : 
```html
<Big> ... </Big>
<Small> ... </Small>

```

To Strike text : 
```html
<S> ... </S>

```

For Paragraph : 
```html

```

For Superscript : 
```html

```
Eg. `X<sup>2</sup>+5`

For Subscript : 
```html

```
Eg. `H<sub>2</sub>O`
