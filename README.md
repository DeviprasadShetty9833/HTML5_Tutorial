# HTML

‚Ä¢ Extension :  `.html`  OR  `.htm`

‚Ä¢ Syntax of Html : 
```html

<html>
   <head>
      <title> TITLE </title>
   </head>

   <body>
      Actual contents in the page
   </body>
</html>

```

‚û¢ Attributes of `<body>` tag :

> - Background - image file as a background behind the displayed text.
> - BGCOLOR - background color of web page.
> - Text - text color 
> - Link - color for hyperlinks.
> - Vlink- color of visited hyperlinks.

‚Ä¢ Angle brackets : `< >`  

‚Ä¢ Start & End tag : `<tag>` & `</tag>`

‚û¢ Tags : Not Case Sensitive 

‚Ä¢ For Comments : 
